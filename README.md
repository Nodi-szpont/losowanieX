# Losowanie liczb i zgadywanie

Aplikacja do losowania liczb i zgadywania z interfejsem 5 przycisków.

## Nauczasz się:
- **Random** - generowanie losowych liczb
- **Handler i Looper** - obsługa operacji asynchronicznych
- **ArrayList** - dynamiczne listy
- **Collections.shuffle()** - tasowanie elementów
- **Button listeners** - obsługa zdarzeń przycisków
- **Color** - zmiana kolorów dynamicznie
- **onDestroy** - czyszczenie zasobów
